API Keys

Add a key for any provider you want to use. One is enough — if you only configure one provider, both cover letter and outreach will use it.

Keys are stored locally and only sent to the provider's API.

Cover Letter

Which model writes your cover letters and in what tone.

Cover Letter Prompt

Keep these tokens in your prompt — each one gets replaced with your real data when the prompt runs:

{resumeText} — your résumé text

{contextText} — your context.md content

{jobDescription} — the job posting on the page

Write tone instructions directly in your prompt — there's no {tone} placeholder in custom mode.
